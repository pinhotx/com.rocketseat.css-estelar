## O que significa CSS
<br>

> Cascading Style Sheet

> Código para criar estilos no HTML

> HTML é uma estrutura, CSS é a beleza

> <strong>NÃO</strong> é uma linguagem de programação

> É uma linguagem de Style Sheet (Folha de Estilo)
